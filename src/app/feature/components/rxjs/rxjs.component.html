8. Take, TakeLast & TakeUntil Operator

The take, takeUntil, takeWhile & takeLast operators allow us to filter out the emitted values from the observable.

take()

The take(n) emits the first n values
takeLast()

takeLast(n) emits the last n values.
takeUntil(condition)

The takeUntil(notifier) keeps emitting the values until it is notified to stop.

retryWhen(), scan() and delay()

RetryWhen - Retry an observable sequence on error based on custom criteria.

scan - Useful for encapsulating and managing state. Applies an accumulator (or "reducer function") to each value from the source after an initial state is established -- either via a seed value (second argument), or from the first value from the source.

delay - Delays the emission of items from the source Observable by a given timeout or until a given Date.

14. ConcatMap Operator | concatMap()


Projects each source value to an Observable which is merged in the output Observable, in a serialized fashion waiting for each one to complete before merging the next.

15. SwitchMap Operator | switchMap()


- Projects each source value to an Observable which is merged in the output Observable, emitting values only from the most recently projected Observable.

- Used to implpement search functionality to cancel the last request


Ex - 01 | Map

Videos : ['Tech','Comedy','News']


Ex - 02 | Map + switchAll

Videos : ['Tech','Comedy','News']


Ex - 03 | switchMap

Videos : ['Tech','Comedy','News']


16. ExhaustMap Operator | exhaustMap()


Projects each source value to an Observable which is merged in the output Observable only if the previous projected Observable has completed.
